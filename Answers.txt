-Foreign Key is a constraint that creates a relationship between two tables.  
  It will Link to another table with primary key as a constraint.
- ON DELETE CASCADE is a sql constraint where if we delete any row in a table and its corresponding table row also will be deleted.
  In this it will deletes all expenses related to a user when that user account is deleted.


-Backend use session based system to verify if the user is log-in or not.  
  Whenever we have to add,delete,modify and edit the expenses, firstly backend will check for the log-in user and stored its user-id through Sql query.
  so, it will not possible for one user to access another user expenses.


- SELECT category, SUM(amount) AS total
  FROM expenses
  WHERE user_id = %s
  GROUP BY category
  ORDER BY total DESC;
